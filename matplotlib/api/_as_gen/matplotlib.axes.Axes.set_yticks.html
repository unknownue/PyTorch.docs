matplotlib.axes.Axes.set_yticks¶

Axes.set_yticks(ticks, *, minor=False)¶

Set the yaxis' tick locations.

Parameters:
tickslist of floats

List of tick locations.

minorbool, default: False

If False, set the major ticks; if True, the minor ticks.
