==================
Simple ImageGrid 2
==================

Align multiple images of different sizes using
`~mpl_toolkits.axes_grid1.axes_grid.ImageGrid`.



.. image:: /gallery/axes_grid1/images/sphx_glr_simple_axesgrid2_001.png
    :alt: simple axesgrid2
    :class: sphx-glr-single-img






.. code-block:: default


    from matplotlib import cbook
    import matplotlib.pyplot as plt
    from mpl_toolkits.axes_grid1 import ImageGrid


    fig = plt.figure(figsize=(5.5, 3.5))
    grid = ImageGrid(fig, 111,  # similar to subplot(111)
                     nrows_ncols=(1, 3),
                     axes_pad=0.1,
                     label_mode="L",
                     )

    # demo image
    Z = cbook.get_sample_data("axes_grid/bivariate_normal.npy", np_load=True)
    im1 = Z
    im2 = Z[:, :10]
    im3 = Z[:, 10:]
    vmin, vmax = Z.min(), Z.max()
    for ax, im in zip(grid, [im1, im2, im3]):
        ax.imshow(im, origin="lower", vmin=vmin, vmax=vmax)

    plt.show()
