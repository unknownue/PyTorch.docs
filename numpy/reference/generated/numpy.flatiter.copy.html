numpy.flatiter.copy¶

method

flatiter.copy()¶

Get a copy of the iterator as a 1-D array.

Examples

>>> x = np.arange(6).reshape(2, 3)
>>> x
array([[0, 1, 2],
       [3, 4, 5]])
>>> fl = x.flat
>>> fl.copy()
array([0, 1, 2, 3, 4, 5])
